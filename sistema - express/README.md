SISTEMA EXCLUSIVO - EXPRESS RESTAURANTES - UNIDADE COLORADO

Como iniciar no Termux:

cd "/storage/emulated/0/Download/sistema - express"
bash ferramentas/instalar_termux.sh
python iniciar.py

Pastas importantes:
- importar/analiticos: coloque PDFs de Requisição Analítica Tekinisa.
- importar/ordens_compra: coloque PDFs ou CSVs de Ordem de Compra.
- importar/backups: coloque backups externos para restauração.
- relatorios/pdf: relatórios em PDF.
- relatorios/excel: relatórios em Excel/CSV.
- backups: backups criados pelo sistema.

Atalhos:
- Digite 999 para voltar/cancelar a operação atual.
- Use Leitura por código para entrada, saída, conferência e vínculo de código de barras.
- Use Produtos > Cadastro automático para cadastrar produtos a partir do Analítico Tekinisa.

Regra de segurança:
Sempre faça backup completo antes de grandes importações ou restaurações.
